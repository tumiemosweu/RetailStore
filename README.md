# RetailStore

### Description
This is Spring Boot Java 11 application, which exposes an API such that given a
bill, it finds the net payable amount.

### Instructions
Clone repo:

```
git clone https://github.com/tumiemosweu/RetailStore.git
```

```
cd RetailStore
```


Run app with maven wrapper:

```
mvn spring-boot:run
```

Running tests:
```
mvn test
```
